Problem

This problem occurs due to wrong configuration in Turbo C++ directories.

 

Solution

Please follow the steps to fix the problem
Step 1. Go in Options menu.
Step 2. Then in Directories...
 
 
Step 3. Edit the directories show in following images
 
Include Directories -> C:\TURBOC3\INCLUDE
Library Directories -> C:\TURBOC3\LIB
Output Directory -> C:\TURBOC3\SOURCE
Source Directories -> C:\TURBOC3\SOURCE                [Wrong in image]
 
 
 
Step 4. Click on OK button
 
Done. Problem Solve.
